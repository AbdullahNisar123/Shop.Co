## ğŸš€ Getting Started

### 1. Clone this repo

```bash
git clone https://github.com/AbdullahNisar123/shopco-ecommerce.git
cd SHOP.CO-ECOMMERCE
```

### 2. Install dependencies (if using Tailwind)

```bash
npm install
```

### 3. Compile Tailwind CSS (optional)

```bash

```

### 4. Run locally
